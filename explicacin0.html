Explicación

El método consiste en comparar los elementos de un arreglo con un elemento elegido como el más pequeño o más grande en un determinado momento. Este elemento, que también forma parte del arreglo, se intercambia con otro elemento para ordenar el conjunto, colocando los elementos más pequeños o más grandes en las primeras posiciones (dependiendo del criterio de ordenamiento) y ubicando cada elemento en orden de izquierda a derecha.

El proceso comienza con el primer elemento del arreglo, el cual se considera el menor de todo el arreglo (Suponiendo que el criterio de ordenamiento es de menor a mayor). Este primer elemento se compara con el segundo elemento para determinar cuál de los dos es más pequeño.

Si el segundo elemento es más pequeño, su posición se almacena en una variable auxiliar (llamada "minimo" para esta explicación). Si el primer elemento es más pequeño, su posición se mantiene en la variable auxiliar.

Posteriormente, se compara el tercer elemento del arreglo con el elemento que se encuentra en la posición indicada por la variable minimo, siguiendo el mismo proceso de comparación y asignación mencionado para los primeros 2 elementos. Este proceso continua hasta llegar al último elemento de la lista, momento en el cual la variable minimo tendrá almacenada la posición del elemento más pequeño del arreglo.

Al finalizar el primer recorrido del arreglo, se deberá intercambiar el contenido de la primera posición del arreglo con el contenido de la posición indicada por la variable mínimo.

El ordenamiento continúa considerando al segundo elemento del arreglo como el siguiente elemento más pequeño de la lista, el cual se comparará con el tercer elemento de la lista para determinar cuál es el elemento más pequeño en la segunda iteración, realizando los mismos pasos mencionados en la primera iteración.

Al llegar al último elemento de la lista, se intercambia el contenido de la segunda posición del arreglo y el contenido de la posición indicada por la variable mínimo, lo cual colocaría al segundo elemento más pequeño de la lista en la segunda posición.

Esta serie de pasos continúa hasta recorrer el arreglo un número "n" de veces (Llegando hasta el último elemento de la lista en el último recorrido), momento en el cual se considerará que el arreglo se encuentra ordenado de menor a mayor.

Explicación de las variables

Nombre de la variable Descripción
V Representa al arreglo o vector que contiene los elementos a ordenar.
n Representa el tamaño del arreglo V.
minimo
Representa la posición en la cual se encuentra el elemento con el menor valor en un determinado momento del ordenamiento.
aux Variable auxiliar la cual permitirá almacenar temporalmente el elemento con menor valor.
i Variable auxiliar la cual representa la posición del elemento el cual se comparará con el resto de los elementos del arreglo con el propósito de encontrar cuál es el que tiene menor valor en una determinada iteración (Comienza con la posición del primer elemento del arreglo hasta llegar a la última posición del arreglo).
j Variable auxiliar que representa la posición a la cual se requiere acceder del arreglo (Comenzado una posición después de i).
