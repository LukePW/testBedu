Resumen

Funcionamiento del algoritmo

El método de ordenamiento burbuja funciona de la siguiente forma:

1. Comparación de pares adyacentes

Se comparan pares de elementos adyacentes dentro del arreglo, es decir, cada elemento se compara con el elemento que tiene a su derecha, comenzando con el primer y segundo elemento del arreglo en cada iteración.

2. Intercambio de elementos

Si los elementos comparados están en el orden incorrecto según el criterio de ordenamiento elegido, entonces se intercambia la posición de ambos elementos.

3. Recorrido del arreglo

El proceso de comparación e intercambio de elementos adyacentes continúa para cada elemento del arreglo hasta llegar al final del mismo.

4. Repetición del proceso

Una vez alcanzado el final del arreglo, se vuelve al inicio del arreglo, donde se volverá a iniciar el proceso de comparación e intercambio para todos los elementos del arreglo.

Al final de cada iteración, se tendrá al siguiente elemento más grande o pequeño entre las últimas posiciones del arreglo (Dependiendo del criterio de ordenamiento elegido)

5. Finalización

El algoritmo finaliza cuando se haya recorrido el arreglo un total de 'n' veces, donde 'n' representa el tamaño del arreglo, momento en el que se considerara que el arreglo está ordenado.

Características

  • Sencillo de implementar.
  • Es más eficiente en arreglos con pocos elementos.
  • Se considera como un algoritmo que aplica comparaciones para ordenar un conjunto de datos por la forma en la cual trabaja.
  • Es eficiente en términos de memoria, ya que el proceso de ordenamiento se realiza en el mismo arreglo en el que se encuentran los elementos a ordenar (No requiere la creación de otro arreglo).
  • No se recomienda para arreglos de gran tamaño.
  • Es considerado como uno de los algoritmos que tardan más tiempo en ordenar un conjunto de datos debido a la forma en la cual trabaja.
