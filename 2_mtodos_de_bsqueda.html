2. Métodos de búsqueda

¿Qué son los métodos de búsqueda?

Son algoritmos que permiten localizar un determinado elemento dentro de un conjunto de datos. La forma en que se realice este proceso dependerá del método elegido para realizar la búsqueda, así como de la estructura en la que estén almacenados los elementos.

Propósito

El propósito principal de aplicar estos algoritmos es verificar si existe un determinado elemento dentro de un conjunto de datos, devolviendo la posición en la que se encuentra en caso de ser necesario. Este proceso se aplica frecuentemente dentro de otras tareas, como el filtrado de datos, la actualización de datos, entre otras.
