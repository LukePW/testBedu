Explicación

El método consiste en comparar un determinado elemento (Algunas veces llamado valor clave) de un arreglo con los elementos que se encuentren a su izquierda, con el propósito de colocar el mismo en la posición que le corresponde, desplazando los elementos más grandes o pequeños a la derecha (Dependiendo del criterio elegido para ordenar el arreglo).

El proceso comienza seleccionando al segundo elemento como valor clave, el cual se comparará con el primer elemento del arreglo para verificar si cumple con el criterio seleccionado para el ordenamiento (Para esta explicación, el criterio es de menor a mayor).

Si el primer elemento es mayor que el valor clave, entonces se intercambia el contenido de ambas posiciones, desplazando al elemento mas grande a la derecha. En caso de que no sea mayor, entonces se conserva el valor de ambas posiciones.

Después, se selecciona al tercer elemento del arreglo como el valor clave para lo que se considera como la segunda iteración del ciclo "para".

Al igual que en la primera iteración, se compara el valor clave con los elementos que se encuentran a su izquierda con el propósito de hallar la posición en la cual el valor debe colocarse (Deteniendo las comparaciones cuando se encuentre un elemento menor al valor clave o cuando se llegue al inicio del arreglo en cada iteración). Durante este proceso, los elementos mayores al valor clave se desplazan una posición a la derecha. Cuando se detienen las comparaciones en una iteración, se realiza una última comparación entre el valor clave y el elemento que detuvo las comparaciones o el primer elemento del arreglo si se llegó al inicio del mismo, esto con el propósito de colocar ambos valores en su posición correcta.

Este proceso continua hasta que todos los elementos del arreglo se encuentren colocados en su posición correcta (Ordenados de menor a mayor).

Explicación de las variables

Nombre de la variable Descripción
V Representa al arreglo o vector que contiene los elementos a ordenar.
n Representa el tamaño del arreglo V.
aux Variable auxiliar la cual permitirá almacenar temporalmente al elemento del cual se busca su posición correcta dentro del arreglo en una determinada iteración (Valor clave)
i Variable auxiliar la cual representa la posición del elemento que se tomará como valor clave para realizar las comparaciones en una determinada iteración. 
p Variable auxiliar la cual representa la posición de uno de los elementos que se encuentran a la izquierda del valor clave.
