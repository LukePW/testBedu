Explicación

El método consiste en buscar un elemento de forma secuencial dentro de un arreglo (Esto es, ir comparando cada elemento del arreglo con el elemento que se está buscando desde la primera posición hasta la última).

Si el elemento que se está comparando tiene el mismo valor que el elemento buscado, entonces se almacena dentro de una variable auxiliar la posición en la cual se encontró al elemento dentro del arreglo. En caso de que el elemento comparado no sea el que se está buscando, entonces se verifica si el siguiente elemento es el elemento buscado.

Si al finalizar el proceso de búsqueda el valor de la variable auxiliar es diferente al asignado inicialmente, entonces se considera que el elemento a buscar se encontró dentro del arreglo, cuya posición está en la variable auxiliar. En caso de que la variable auxiliar tenga el mismo valor que inicialmente se le asignó, entonces se considera que el elemento a buscar no se encontró durante la búsqueda.

Explicación de las variables

Nombre de la variable Descripción
V Representa al arreglo o vector en el cual se buscará el elemento.
encontrado Variable auxiliar la cual permitirá conocer si se ha encontrado el elemento a buscar dentro del arreglo, almacenando la posición en que se encontró el elemento en caso de existir.
i Variable auxiliar que representa la posición del arreglo a la cual se accederá para verificar si el elemento a buscar se encuentra en esa posición.
