Explicación

El método consiste en comparar pares de elementos adyacentes dentro de un arreglo según un determinado criterio (Es decir, se compara un elemento con el elemento que se encuentra después de el mismo dentro del arreglo).

Si al comparar ambos elementos se determina que están en el orden incorrecto según el criterio establecido, entonces se intercambian sus posiciones en el arreglo, cumpliendo así con el criterio.

Este proceso continúa hasta que todos los elementos del arreglo estén ordenados de acuerdo con el criterio establecido.

Explicación de las variables

Nombre de la variable Descripción
V Representa al arreglo o vector que contiene los elementos a ordenar.
tam Representa el tamaño del arreglo V.
aux Variable auxiliar la cual permitirá almacenar temporalmente el elemento a intercambiar.
i Variable auxiliar que permitirá recorrer todo el arreglo según su tamaño  (El número de veces que se revisará todo el arreglo es igual a su tamaño)
j Variable auxiliar la cual representa la posición del elemento que se está comparando en un determinado momento.
