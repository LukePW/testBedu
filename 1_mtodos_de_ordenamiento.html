1. Métodos de ordenamiento

¿Qué son los métodos de ordenamiento?

Son algoritmos que permiten organizar los elementos de un arreglo o vector de acuerdo a un criterio específico. Este proceso se realiza frecuentemente mediante una comparación entre 2 elementos de un arreglo, la cual se efectúa según el criterio elegido para el ordenamiento.

Propósito

El propósito principal de aplicar estos algoritmos es facilitar otro tipo de operaciones, como puede ser la búsqueda de algún elemento dentro de un arreglo, el cual, en algunas ocasiones, debe estar ordenado para poder aplicar ciertos métodos de búsqueda específicos que requieren que los elementos estén ordenados.

Temas que se abordarán

Los temas que se abordarán en esta sección son los siguientes:

  2. Ordenamiento por Selección
