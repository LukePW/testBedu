Explicación

El método consiste en reducir el rango de búsqueda a la mitad en cada iteración que se realice durante la búsqueda.

Para que este proceso pueda realizarse, es necesario tener 2 variables auxiliares las cuales indicaran el rango que se considerara dentro del arreglo para realizar la búsqueda. En esta explicación, a una de las variables la llamaremos 'primero', la cual representara el inicio del rango que se considerara para realizar la búsqueda, mientras que la segunda variable, la cual llamaremos 'final', representara la última posición que se considerara.

También se utilizará una variable auxiliar (La cual para esta explicación se llamará 'encontro') la cual indicará si el elemento buscado fue encontrado al realizar la búsqueda, la cual debe tener un valor inicial igual a falso.

Adicionalmente, se debe tener otra variable auxiliar para conocer la posición en la cual se encuentra el elemento a buscar (En caso de que exista dentro del arreglo).

El proceso comienza sumando el valor de la variable 'primero' (La cual debe tener un valor inicial igual a 0) y 'final' (Con valor inicial igual a la última posición del arreglo). Posterior a esto, se divide el resultado de la suma entre 2, lo cual indicará la posición en la que se encuentra la mitad del arreglo, posición que se deberá guardar dentro otra variable auxiliar la cual llamaremos 'mitad' (En caso de que el resultado obtenido de la división tenga valores decimal, el mismo deberá redondearse hacia arriba).

Después, se verifica si el elemento que se encuentra en la posición indicada por la variable 'mitad' es igual al elemento que se está buscando. Si el elemento es igual al buscado, entonces se asigna a la variable auxiliar 'encontro' el valor verdadero, lo cual indicará que el elemento ha sido encontrado durante la búsqueda en la posición que indica la variable 'mitad'.

En caso de que el elemento no sea igual al que se busca, entonces se modifica el valor de las variables primero y final según en los siguientes criterios:

a) Si el elemento comparado es mayor al elemento que se está buscando, entonces se debe restar en una unidad el valor de la variable 'mitad', asignando el resultado a la variable 'final'.

b) En caso de que el elemento comparado sea menor al elemento que se busca, entonces se debe sumar en una unidad el valor de la variable 'mitad', asignando el resultado de la operación a la variable 'primero'.

Posterior a la modificación de las variables mencionadas, se deben volver a sumar los valores de ambas variables y posteriormente se debe dividir el resultado de la suma entre 2 para después asignar el resultado de la división en la variable mitad con el propósito de verificar si el elemento almacenado en la posición indicada por la variable mitad es el elemento que se está buscando. El proceso continúa mientras la variable encontro sea igual a falso y el valor de la variable primero sea menor o igual al valor de la variable final.

Si al finalizar el proceso de búsqueda el valor de la variable booleana 'encontro' es verdadero, entonces se considera que el elemento buscado fue encontrado dentro del arreglo, cuya posición se encuentra almacenada dentro de la variable 'mitad'. En caso de que el valor de la variable booleana sea igual a falso (valor inicial), entonces se considerará que el elemento no se encontró en el arreglo.

Explicación de las variables

Nombre de la variable Descripción
V Representa al arreglo o vector en el cual se buscará el elemento.
primero Representa el inicio del rango en el que se buscará al elemento dentro del arreglo en un determinado momento.
final Representa el final del rango en el que se buscará al elemento dentro del arreglo en un determinado momento.
mitad Representa la mitad del rango definido por las variables primero y final, cuyo valor permitirá conocer en qué posición del arreglo se debe verificar si existe el elemento a buscar en un determinado momento.
encontro Variable auxiliar que permitirá conocer si se encontró al elemento buscado.
